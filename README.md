# 🔍 CAMPUS-TRACE

A full-stack Lost and Found web application for college campuses.

## 🛠️ Tech Stack
- **Frontend:** React + Vite, deployed on Vercel
- **Backend:** Node.js + Express, deployed on Railway
- **Database:** MongoDB Atlas
